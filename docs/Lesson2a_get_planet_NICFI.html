Access and mosaic Planet NICFI monthly basemaps¶

A guide for accessing monthly Planet NICFI basemaps, selecting data by a defined AOI and mosaicing to produce a single image. You will need a configuration file in your “My Drive” folder containing the API access key you should have receieved once you signed up for and subscribed to the Planet NICFI program. Please visit https://www.planet.com/nicfi/ to sign up if you have not already.
